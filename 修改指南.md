# 网站内容修改指南

## 1. 基本信息修改

### 1.1 公司名称和Logo
**文件**: `index.html`
**位置**: 第36-40行
```html
<a href="#" class="logo">
    <i class="fas fa-microchip"></i>
    <span>我的公司</span>
</a>
```

**修改方法**:
1. 将`<span>我的公司</span>`改为您的公司名称
2. 如果要使用图片Logo，替换整个`<a>`标签：
```html
<a href="#" class="logo">
    <img src="assets/images/logo.png" alt="您的公司名称" width="200" height="60">
</a>
```

### 1.2 页面标题
**文件**: `index.html`
**位置**: 第7行

将`我的公司官网`改为您的公司名称，例如：
```html

```

## 8. 颜色主题修改

### 8.1 主要颜色变量
**建议**: 在`style.css`文件开头添加颜色变量

**文件**: `style.css`
**位置**: 在文件开头添加
```css
:root {
    --primary-color: #1a365d;      /* 主色 - 深蓝色 */
    --secondary-color: #4299e1;    /* 辅助色 - 浅蓝色 */
    --accent-color: #ff6b35;       /* 强调色 - 橙色 */
    --text-color: #333;            /* 正文颜色 */
    --light-bg: #f8f9fa;           /* 浅色背景 */
    --dark-bg: #1a365d;            /* 深色背景 */
}
```

### 8.2 批量修改颜色
修改以下CSS属性来改变网站颜色：

1. **导航栏颜色**:
```css
.header {
    background-color: var(--primary-color);
}
```

2. **按钮颜色**:
```css
.btn-primary {
    background-color: var(--accent-color);
}
```

3. **链接颜色**:
```css
.nav-link:hover {
    color: var(--secondary-color);
}
```

## 9. 图片替换指南

### 9.1 图片准备
1. **Logo图片**: PNG格式，透明背景，建议尺寸 200×60像素
2. **主视觉图片**: JPG格式，建议尺寸 1920×600像素
3. **产品图片**: JPG格式，建议尺寸 400×300像素
4. **团队照片**: JPG格式，建议尺寸 800×600像素

### 9.2 图片放置
将所有图片放在 `assets/images/` 文件夹中

### 9.3 修改图片引用
**示例**: 修改Logo图片
```html
<!-- 修改前 -->
<i class="fas fa-microchip"></i>

<!-- 修改后 -->
<img src="assets/images/logo.png" alt="公司Logo">
```

## 10. 响应式测试

### 10.1 测试方法
1. 在浏览器中打开网站
2. 按F12打开开发者工具
3. 点击"切换设备工具栏"图标
4. 选择不同设备尺寸测试

### 10.2 常见调整
如果在小屏幕上显示不正常，可以：
1. 调整字体大小
2. 修改边距和间距
3. 改变布局方式

1. **备份原文件**: 修改前先复制一份原文件
2. **逐步修改**: 一次只修改一个地方，测试后再继续
3. **检查链接**: 确保所有链接都正确
4. **测试响应式**: 在不同设备上测试显示效果
